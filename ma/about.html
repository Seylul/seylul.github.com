About

This application about the find the shortest route or the fastest route that how you want. You can write destination that you don't want to go. So the route will calculate as you want. Application will want to you only orijin,destination and expected route(don't want destination). Application is using for this Floyd-Warshall Algorithm. This application is designed with JQuery-Mobile/Codiqa. And it use the Google Maps/ GoogleMaps-Traffic.
